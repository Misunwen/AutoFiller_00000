🎟️ 網頁自動填表小幫手 (AutoFiller) - 使用說明書
🌟 核心功能簡介
這是一個專為特定網頁（如：購票系統）打造的 Chrome 瀏覽器加速小工具。 它可以幫你省去手動點擊的時間，達到「進網頁即填妥」的極速體驗！

✅ 自動勾選同意條款：自動尋找並勾選「我同意」之類的核取方塊。
🔢 自動選擇票數/數量：自動鎖定下拉選單，並填入你預設好的數量 (0~4)。
⚡ 即時連動與記憶：在面板更改設定的瞬間，網頁立刻跟著變動，且設定會永久記憶，下次進網頁直接套用！
🛠️ 如何安裝到 Chrome 瀏覽器？
由於此插件是私人開發版本（未上架 Chrome 商店），請透過以下「開發者模式」安裝：

準備資料夾：確認你的電腦裡有一個包含 manifest.json、popup.html、popup.js、content.js 四個檔案的資料夾（例如命名為 AutoFiller）。
開啟擴充功能頁面：打開 Chrome 瀏覽器，在網址列輸入 chrome://extensions/ 並按下 Enter。
開啟開發人員模式：在頁面的「右上角」，將「開發人員模式」的開關打開。
載入插件：點擊頁面「左上角」的「載入未封裝項目」按鈕。
選擇資料夾：選取你的 AutoFiller 資料夾，點擊確認。
📌 釘選插件 (推薦)：點擊 Chrome 右上角的「拼圖圖示」，找到「網頁自動填表小幫手」，點擊旁邊的「圖釘📌」將它固定在工具列上，方便隨時操作。
🎮 操作指南 (日常使用)
本插件主打「無腦操作」，沒有繁瑣的儲存按鈕：

預先設定：點擊 Chrome 右上角工具列的「插件圖示」。
調整選項：
勾選 「開啟自動打勾」。
在下拉選單 「選擇數量」 (例如：2 張)。
✨ 瞬間生效：當你點選的瞬間，設定就已經自動存檔了！如果此時你剛好在目標網頁上，你會看到網頁上的數量和打勾框瞬間跟著改變。
🔥 搶票/實戰最高級技巧 (Pro-Tips)
因為這個插件內建了 「網頁載入自動偵測機制 (0.1秒極速連點)」，你可以這樣用來搶票：

開賣前 5 分鐘：先點開插件，設定好你要的張數（如：4張），並開啟自動打勾。
時間到，重新整理：當開賣時間一到，你只要按下 F5 (重新整理網頁)。
享受全自動：網頁載入出來的瞬間（只要下拉選單一浮現），插件會在 0.1秒內自動幫你選好 4 張並打好勾！
你只需要做：肉眼確認數量正確，輸入驗證碼，點擊送出！大幅縮短黃金搶票時間！
⚠️ 常見問題與注意事項
Q: 為什麼去別的網站沒有反應？
A: 因為這個插件是「特製版」，程式碼 (content.js) 裡面鎖定了特定的網頁標籤 ID (TicketForm_agree 等)。它只會在擁有這些 ID 的特定網站上運作，不會影響你日常瀏覽其他網站。
Q: 為什麼有時候下拉選單沒有自動選上？
A: 如果系統大當機、網頁跑到一半卡住，導致下拉選單根本沒有載入出來，插件就無法選擇。只要重新整理網頁，插件就會再次嘗試自動填寫。
Q: 如何暫時停用它？
A: 只要點開插件面板，把「開啟自動打勾」取消勾選，並把數量選回「不更改」即可。
